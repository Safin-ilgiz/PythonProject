# Проект: Виджет банковский операций клиента

# Цель проекта:
Позволяет фильтровать исортировать банковские данные, а также маскировать номера карт и счетов

# Примеры работы функций
- "filter_by_state(list_dict, state)"
- "sort_by_date(list_dict, reverse_order= True)"

# Новые функции и генераторы (Тема 11)

В проект добавлены инструменты для эффективной фильтрации и обработки данных:
1. `filter_by_currency` — функция-генератор, фильтрующая транзакции по заданному коду валюты (например, USD).
2. `transaction_descriptions` — генератор, который поочередно возвращает текстовое описание для каждой операции.
3. `card_number_generator` — генератор номеров банковских карт, выдающий номера в строго заданном диапазоне и форматирующий их по шаблону `XXXX XXXX XXXX XXXX`.

Все новые генераторы полностью покрыты параметризованными тестами на Pytest.
